## Key Responsibilities

### Mainframe Services Engineering
- Design and maintain CICS transaction programs and Db2 stored procedures
- Develop JCL jobs, REXX utilities, and batch processing flows
- Work with VSAM files, GDGs, and MQ messaging
- Optimize Db2 queries and tune system performance
- Manage copybooks and data structures

### API Gateway & Mediation (IBM DataPower)
- Configure and maintain Multi-Protocol Gateway (MPGW) and Web Service Proxy services
- Implement security policies: AAA, JWT/OAuth2, MTLS, WS-Security
- Develop transformations using XSLT and GatewayScript
- Set up rate limiting, threat protection, and schema validation
- Manage high availability and firmware upgrades

### Mainframe-to-Cloud Integration
- Expose CICS and Db2 services as REST APIs using z/OS Connect EE
- Integrate with AWS API Gateway and TYK
- Build Java/Spring Boot microservices that interact with mainframe
- Implement idempotency, circuit breakers, and timeout management
- Ensure end-to-end distributed tracing with correlation IDs

### Data Platform Modernization
- Migrate Db2 tables to Aurora PostgreSQL Global Database
- Design schemas, DDL, and migration scripts
- Configure multi-region readers/writers, proxies, and failover
- Perform DR drills and establish performance baselines
- Work with CockroachDB for distributed SQL use cases


### Reliability & DevOps
- Create SRE-style runbooks and incident response procedures
- Implement canary and blue/green deployment strategies
- Monitor performance metrics (TPS, P95 latency, resource utilization)
- Manage TLS/SSL certificate lifecycle across all platforms
- Participate in on-call rotation for 24/7 payment systems

### Security
- Implement RACF security and dataset permissions
- Integrate AWS Secrets Manager and KMS
- Configure OAuth2/OIDC token exchange
- Ensure encryption at rest and in transit
- Maintain audit trails and compliance logging
